Cross-Border Marketplace Business Plan (Myanmar)

ဘာသာစကား: မြန်မာ  
Version: MVP v1.0  
  

1\. Product Vision
==================

Facebook Marketplace, Preorder Page နှင့် Cargo Service များကို Platform တစ်ခုအောက်တွင် ပေါင်းစည်းပြီး ယုံကြည်စိတ်ချရသော Cross-Border Marketplace တစ်ခု တည်ဆောက်ရန်။

2\. Business Goals
==================

• Verified Seller Ecosystem  
• Transparent Cargo Tracking  
• Authentic Products  
• Competitive Pricing  
• Auto Approval Workflow  
• Local + Cross-border Trading  
• AI Assisted Seller Tools

3\. Target Users
================

Admin - System Management  
Sales Person - Verified Seller  
Client - Buyer

4\. Core Product Types
======================

• In Stock  
• Cargo Item  
• Promotion Opportunity (Pre-buy)  
• Preorder  
• Local Marketplace  
• Used Item (Future)

5\. End-to-End Workflow
=======================

Seller Registration → Admin Verification → Product Upload → Product Approval (Auto/Manual)  
→ Customer Browse/Search → Cart → Checkout → Payment  
→ Purchase / Packing → Cargo → Warehouse → Delivery  
→ Review & Rating

6\. Cargo Tracking Flow
=======================

Order Placed  
Payment Confirmed  
Waiting Purchase  
Purchased  
Seller Received  
Packed  
Bangkok Warehouse  
Export Clearance  
Air Cargo  
Myanmar Customs  
Yangon Warehouse  
Out for Delivery  
Delivered

7\. Tech Stack
==============

Frontend: Next.js 15, React, TypeScript, TailwindCSS, shadcn/ui  
Backend: NestJS, PostgreSQL, Prisma  
Cache: Redis  
Queue: BullMQ  
Storage: MinIO / S3  
Search: Meilisearch  
Auth: Auth.js + OTP + JWT  
Realtime: Socket.IO  
Mobile (Phase-2): React Native Expo  
Deployment: Docker, Coolify, Cloudflare
