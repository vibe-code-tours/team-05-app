# Glossary

> **Buy Anywhere. Track Everything. Trust Every Seller.**

---

## Business Terms

### Cross-Border

Commerce that takes place between buyers and sellers in different countries. In the context of this platform, cross-border primarily refers to trade between Myanmar and its neighboring countries (Thailand, China, Japan) as well as other international markets.

### Cargo

The physical goods and packages being transported from sellers to buyers. In this platform, cargo tracking is a core feature that provides real-time visibility into the location and status of shipments throughout the delivery journey.

### Preorder

A purchasing model where customers place orders for products that are not yet in stock or available for immediate delivery. Preorders are common in cross-border commerce where products must be sourced internationally and shipped to Myanmar. The platform supports preorder listings with estimated delivery timelines.

### Promotion Opportunity

A marketplace feature that allows sellers to participate in time-limited promotional events, flash sales, or featured campaigns. Promotion opportunities increase product visibility and are available for a fee, forming part of the platform's revenue model.

### Verified Seller

A seller who has completed the platform's structured verification process, which may include identity verification, business documentation review, product authenticity checks, and reputation assessment. Verified sellers receive trust badges and enhanced visibility on the platform.

### Premium Seller Membership

A tiered subscription model that provides sellers with enhanced platform features such as lower commission rates, priority support, advanced analytics dashboards, and exclusive marketplace tools.

## Technical Terms

### RBAC (Role-Based Access Control)

A security mechanism that restricts system access based on the roles assigned to users. In this platform, RBAC defines what different user types (customers, sellers, administrators, cargo partners) can see and do within the system.

### OTP (One-Time Password)

A temporary, single-use verification code sent to a user's phone or email. Used during login, registration, and sensitive operations to verify user identity and enhance security.

### JWT (JSON Web Token)

A compact, URL-safe token format used for securely transmitting information between parties. In this platform, JWTs are used for authentication and authorization, carrying user identity and permission data in API requests.

### BullMQ

A Node.js-based message queue library built on Redis. Used in this platform for managing background jobs such as sending notifications, processing orders, generating reports, and handling asynchronous tasks that should not block the main application.

### Prisma

A modern Object-Relational Mapping (ORM) tool for Node.js and TypeScript. Used in this platform to define database schemas, run migrations, and interact with the database through type-safe queries.

### NestJS

A progressive Node.js framework for building efficient, reliable, and scalable server-side applications. This platform's backend API is built on NestJS, leveraging its modular architecture, dependency injection, and TypeScript support.

### Next.js

A React-based framework for building full-stack web applications. This platform's customer-facing and seller-facing web interfaces are built on Next.js, providing server-side rendering, static generation, and optimized performance.

### Cloudflare R2

An S3-compatible object storage service provided by Cloudflare. Used in this platform for storing product images, seller documents, and other static assets with global CDN delivery and no egress fees.

### Coolify

An open-source, self-hosted platform-as-a-service (PaaS) solution. Used in this platform for deployment, container management, and infrastructure orchestration, providing a simplified alternative to complex cloud infrastructure management.

## Platform Terms

| Term | Definition |
|------|------------|
| **Marketplace** | The platform itself where buyers and sellers interact to complete transactions. |
| **Seller Dashboard** | The seller-facing interface for managing products, orders, inventory, analytics, and account settings. |
| **Customer Portal** | The buyer-facing interface for browsing products, making purchases, and tracking orders. |
| **Admin Panel** | The administrator interface for managing users, verifying sellers, monitoring platform health, and configuring settings. |
| **Trust Badge** | A visual indicator displayed on verified seller profiles and listings, signifying that the seller has completed platform verification. |
| **Order Tracking** | The real-time visibility feature that shows customers the current status and location of their cargo throughout the delivery journey. |
