# Admin Panel Skill

## Admin Features

### Dashboard
- Total orders, revenue, users
- Recent activity
- System health

### User Management
- List/search users
- View user details
- Change user roles
- Ban/unban users

### Product Management
- Pending product approvals
- Approve/reject products
- Featured products
- Category management
- Brand management

### Order Management
- All orders view
- Manual order status update
- Refund processing
- Dispute resolution

### Content Management
- Banner management
- Promotion management
- Coupon management
- Notification broadcasts

---

## Rules

**Always:**
- Log all admin actions
- Require confirmation for destructive actions
- Show audit trail
- Protect admin endpoints

**Never:**
- Allow non-admin access
- Skip action logging
- Allow bulk delete without confirmation
